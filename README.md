# 놀이 수업터

전자칠판에서 바로 사용할 수 있는 놀이형 학습 게임 보관함입니다.

## 실행 방법

`index.html` 파일을 크롬, 엣지 등 웹 브라우저로 열면 바로 실행됩니다.

## GitHub Pages 배포

이 프로젝트는 정적 HTML, CSS, JavaScript만 사용하므로 GitHub Pages에서 바로 실행할 수 있습니다.

1. GitHub에 새 저장소를 만듭니다.
2. 이 폴더의 `main` 브랜치를 GitHub 저장소에 push합니다.
3. GitHub 저장소의 `Settings` → `Pages`에서 `Deploy from a branch`를 선택합니다.
4. Branch를 `main`, 폴더를 `/root`로 설정하고 저장합니다.
5. 잠시 뒤 표시되는 Pages 주소로 접속합니다.

## 현재 포함된 게임

### 규칙 탐험대

- 과목: 수학
- 학년: 4학년
- 단원: 4학년 1학기 5단원 규칙 찾기
- 바탕 자료: `3~4학년군_수학(박)4-1_단원학습지도안_5단원.hwp.pdf`

지도안의 단원 전개를 바탕으로 다음 내용을 문제에 반영했습니다.

- 도형 배열에서 규칙을 찾아 수로 나타내기
- 복잡한 도형 배열의 규칙을 식으로 나타내기
- 수 배열표에서 규칙 찾기
- 등호의 의미와 등호가 있는 식 판단하기
- 덧셈식과 뺄셈식의 배열에서 규칙 찾기
- 곱셈식과 나눗셈식의 배열에서 규칙 찾기
- 실생활 속 수 배열로 규칙적인 계산식 만들기
- 수 카드로 여러 가지 규칙 만들고 설명하기

## 수업 흐름

1. 게임 보관함에서 게임을 선택합니다.
2. 학급 이름과 학생 명단을 확인합니다.
3. 한 판에 참여할 학생 수를 2명, 3명, 4명 중 선택합니다.
4. 10단계, 8단계, 6단계 중 단계 수를 고르고 입력 시간을 정합니다.
5. 참가자를 무작위로 추첨합니다.
6. 전자칠판에서 각 학생이 자기 칸의 정답을 누릅니다.
7. 모두 답하거나 시간이 끝나면 정답과 입력 시간을 판정해 점수에 반영합니다.
8. 판정 결과와 현재 점수를 각 학생 칸에 5초 동안 보여준 뒤 다음 단계로 자동 이동합니다.
9. 모든 단계가 끝나면 점수 기록과 등수를 확인합니다.

## 점수 규칙

- 정답 기본 점수: 10점
- 시간 보너스: 제한 시간에서 입력까지 걸린 시간을 뺀 만큼 추가 점수
- 오답 또는 시간 초과: 0점
- 학생이 답을 고른 뒤에는 그 학생의 답 버튼이 비활성화됩니다.
- 마지막 5초에는 화면 흔들림과 카운트 소리로 남은 시간을 알려줍니다.
- 학생별 입력 칸은 서로 다른 색으로 구분되고, 선택한 답은 별도 색으로 표시됩니다.

## 새 게임 추가 방법

`app.js`의 `GAME_LIBRARY` 배열에 새 게임 객체를 추가하면 홈 화면에 자동으로 카드가 생깁니다.

필수 정보는 `id`, `title`, `subject`, `grade`, `unit`, `content`, `description`, `tags`, `problems`입니다. 문제는 `type`, `text`, `answer`, `options`, `rule` 형식으로 넣으면 됩니다.
